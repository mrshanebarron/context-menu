# Laravel Design Context Menu

Right-click context menu component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/context-menu
```

## Usage

### Livewire Component

```blade
<livewire:ld-context-menu />
```

### Blade Component

```blade
<x-ld-context-menu />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-context-menu-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-context-menu-views
```
